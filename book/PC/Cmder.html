PowerShell • Alt + Enter : Fullscreen • Ctrl + Alt + u : Traverse up in directory structure (lovely feature!) • End, Home, Ctrl : Traverse text as usual on Windows • Ctrl + r : History search • Shift + mouse : Select and copy text from buffer • Right click / Ctrl + Shift + v : Paste text
